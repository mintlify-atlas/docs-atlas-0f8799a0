## What is Yagua?

Yagua is a free, open-source desktop image processor built for developers, designers, and photographers who need to process images in bulk without browser limitations or paid subscriptions. Everything runs locally on your machine — no internet required (after the initial AI model download).

It combines an intuitive GUI built with CustomTkinter with powerful image processing backends including Pillow, rembg, and EasyOCR, giving you professional-grade tools in a single application.

## Why Yagua?

Most web-based image tools are limited — restricted file counts, low resolution caps, and features locked behind subscriptions. Yagua runs everything locally on your machine:

- **No file size limits** — process any resolution image
- **No internet dependency** — everything runs offline
- **No subscriptions** — completely free and open source (MIT license)
- **Batch processing** — handle up to 100 images per operation
- **Multi-language** — available in Spanish, English, and Portuguese
